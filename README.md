# Yara-rules
are text-based, open-source patterns used by cybersecurity professionals to identify and classify malware by searching for specific text, binary strings, or behavioral characteristics in files, memory, or network traffic. They serve as a "Swiss army knife" for threat hunting and incident response, allowing analysts to create custom detection rules based on malicious code, file signatures, or metadata, with syntax similar to the C programming language.

Definition:
A rule-based language used to detect malware in files or memory.

What it analyzes:

    File content

    Binary patterns

    Strings inside executables

Use case:

    Detect malicious DLLs, EXEs, or in-memory payloads

 Key idea:
“What is this file?”
